# ByteSketch

A modern, space-themed landing page built with Next.js and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Space-themed design with pixelated elements
- Smooth animations and transitions
- Responsive layout
- Modern UI components
- Interactive elements
